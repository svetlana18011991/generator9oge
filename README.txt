3. offline_assets_bundle.js — ДОБАВИТЬ рядом с index.html.

Что делает патч:
- картинки из присланного архива переведены в base64;
- скачанные презентации получают фоны через data:image и открываются с фонами без интернета;
- стикеры для печатного файла берутся через window.getEmbeddedAssetUrl и тоже попадают в скачанный HTML как data:image;
- превью стилей презентации в меню тоже берёт картинки из base64.

Важно:
- после замены файлов нажать Ctrl+F5 на странице генератора;
- затем заново сгенерировать вариант/презентацию/файл для печати;
- MathJax и Google-шрифт Caveat всё ещё подключаются с CDN, поэтому без интернета формулы могут отображаться как LaTeX, но фоны и стикеры будут на месте.
